# Where to host NAS

I am comparing two options in this section: Cloud and Local. I will start by computing their respective cost (subscription or buying material + electricity)

Cloud subscription:
I use Hetzner for the cloud subscription as it is the cheapest I could find while being known to be a reliable provider (in all forums i could read, everybody was satisfied with hetzner).
I suppose that a NAS system would cost 539 euros (62*2 1TB HDD + 375 Syonlogy DS224 + 40 4GB DDR4 RAM) in material and the electricity costs will be based on France which is 0.194euros per KWh. I assume that the NAS local server will run 24/7 with a IDLE and ACCESS consumption of 4W and 16W respectively. Since it will be mostly IDLE i will take the 4W as the reference which is clearly a lower bound since it does not account for HDD access and I want to run other applications in addition to NAS. 


Cloud Cost:
- 1TB Hetzner Storage Box (BX11): 3.84 euros/month
- VM - 2vCPUs + 4GB RAM + 40GB SSD (CX23): 3.59 euros/month
- ipv4 address: 0.60 euros/month
- price evolution in the future: will ignore it for now
- Annual Cost: 96.36 euros/year

Local Cost:
- 1st batch of material: 539 euros
- Electricity: 6.93 euros/year
- Replacing/Upgrading material: will ignore it for now
- Evolution of electricity prices: will ignore it for now
- Annual Cost (after purchase): 6.93 euros/year

Local cost becomes advantageous after 6 years. This can be considered the absolute minimum since this is IDLE consumption and hardware replacement was ignored.

But there are advantages to using cloud.
- No hardware management. buy and enjoy.
- No regret if i'm not satisfied while local setups is an investment. It's good for first tests.
- My hardware requirements can evolve in the future. No problem in Cloud.
- Storage Box in Hetzner are backed by RAID which can stand several disk failures (https://www.hetzner.com/storage/storage-box/) while the local low-cost setup has only RAID 1.
- Privacy of data in the Cloud is not a problem since it can all be encrypted with no data visible at any moment in the server.
- Scaling is easy and straightforward.


Conclusion: Go for Cloud as a first step at least for the 1st year.
